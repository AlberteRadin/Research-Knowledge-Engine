# Research Knowledge Engine

This repository contains the source code for the Research Knowledge Engine, composed of a **backend** and **frontend** to handle large-scale academic data.

## Structure

- `backend/` : Python FastAPI service with AI functionality.
- `frontend/` : React-based user interface.
- `docker-compose.yml` ensures coordinated backend/frontend services.

---
Built using **FastAPI**, **ReactJS**, and **Docker Compose**.
